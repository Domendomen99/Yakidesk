# **App Name**: Yakidesk

## Core Features:

- Login Page: Display a login page with Google Sign-In button, logo, and welcoming text.
- Dashboard Layout: Create a dashboard layout with a fixed header and a main content area for desk booking.
- Header Component: Implement a header with the app logo/name and a user avatar dropdown menu.
- Date and Time Slot Selection: Allow users to select a date and time slot for booking using a calendar and time slot tabs.
- Desk Map: Show a grid of desks with status indicators (available/booked) and tooltips for details.
- Booking Confirmation Dialog: Display a modal dialog to confirm booking details and allow time slot selection.
- Simulated Data: Manage booking data using React's useState, and initialize with seed data in a module.

## Style Guidelines:

- Background color: Dark background with a hue of 220, 13% saturation, and 10% lightness (#16181A).
- Primary color: Purple with a hue of 330, 80% saturation, and 60% lightness (#F23399).
- Accent color: A vibrant spring green, analogous to lime green, with a hue of 140, 100% saturation, and 35% lightness (#00B329).
- Headline font: 'Poppins' (sans-serif) for titles.
- Body font: 'Roboto Mono' (monospace) for the main text.
- Use a minimal SVG icon representing a stylized tigella (eight-petal flower). The stroke uses `currentColor` to adapt to the theme.
- Dark mode theme with a professional and modern look.
